ubuntu 20.04 root 远程ssh

horysk

Linux

发布于：2017年09月18日

更新于：2017年09月18日

  • Linux

安装ssh

apt-get install ssh

设置允许root远程登录

vi /etc/ssh/sshd_config 找到 PermitRootLogin prohibit-password 然后用#注释掉这一句 加上 PermitRootLogin yes

重启ssh服务

/etc/init.d/ssh restart

设置root密码

sudo passwd root 设置root密码

现在可以用 winSCP或者ssh远程工具进行连接了。
